# Storyline Scripts

Ce projet sert à documenter et archiver mes scripts JavaScript personnalisés pour Articulate Storyline.

Ces scripts ont été développés au fil des dernières années pour :

- améliorer la qualité visuelle des productions ;
- offrir davantage de personnalisation ;
- étendre certaines fonctionnalités natives de Storyline.

## Organisation

Le dépôt est structuré avec **un dossier par script**.

Chaque dossier peut contenir, selon les besoins:

- le script principal;
- des notes d'utilisation;
- des exemples d'intégration ;
- des variantes ou évolutions du script.

L'objectif est de conserver une base claire, réutilisable et facile à maintenir pour mes futurs projets Storyline.

## Compatibilité

Comme Articulate Storyline évolue au fil des versions, il n'est pas garanti que tous les scripts de ce dépôt fonctionnent encore à l'heure actuelle sans adaptation.
